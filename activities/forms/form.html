Engagement Questions
  1. What is JavaScript? How did it get its name?

    2. Answer: avaScript is a programming language used to create interactive and dynamic content on websites.

  2. What are the two ways you can run JavaScript?

    3. Answer: Netscape partnered with Sun Microsystems (the company behind Java) and rebranded it as JavaScript to take advantage of Javas popularity at the time.

  3. What was the most popular front-end js library?

    4. Answer: Currently, React.js, developed by Meta (formerly Facebook), is the most popular front-end JavaScript library.

  4. What is the most popular front-end js library today?

    5. Answer: For many years, jQuery was the most popular front-end JavaScript library. It simplified DOM manipulation, event handling, and AJAX requests, making it easier to work with web pages.

  5. The form tag has an "action" attribute. What does it do?

    6. Answer: tag specifies the URL where the form data will be sent after submission.

  6. The form tag has a "method" attribute. What does it do?

    7. Answer: attribute defines how form data is sent to the server. The two most common values are post and get

  7. What is the difference between a "POST" and a "GET"?

    8. Answer: A GET request sends data in the URL, like when you search for something on Google and see the search terms in the address bar. A POST request sends data in the background, like when you log in to a website, and your password is sent securely without showing in the URL.

  8. What is AJAX?

    9. Answer: AJAX (Asynchronous JavaScript and XML) is a technique used in web development that allows web pages to update content asynchronously without having to reload the entire page.

  9. Why do you put your JS Code in a separate file?

    10. Answer: JavaScript is placed in a separate file to improve maintainability, reusability, performance, and debugging, while keeping the HTML code clean and organized.
