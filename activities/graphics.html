Questions

  1. What kind of photos are good to encode using JPEG?

    2. Answer: JPEG is best for photographs and images with many colors and gradients.

  2. What kind of photos are good to encode using GIF?

    3. Answer: Simple images with few colors, like logos, icons, and animations.

  3. What is unique about the PNG format?

    4. Answer: It supports lossless compression and transparency, making it ideal for detailed graphics.

  4. How do you resize your photo to the size it should be displayed?

    5. Answer: Use image editing software to set the correct dimensions and optimize file size.

  5. Why resize photos with software instead of CSS?

    6. Answer: Software reduces file size for faster loading, while CSS only scales the image visually.
