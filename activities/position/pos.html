  1. What is normal flow in HTML?

    2. Answer: The default way elements are positioned on a web page such as block elements stack vertically, inline elements flow horizontally.

  2. What does float do? Why do you need to clear the float?

    3. Answer: Moves an element to the left or right, allowing text or other elements to wrap around it. While clearing float prevents elements from being affected by floating elements, ensuring proper layout.

  3. What do absolute, fixed, and relative do?

    4. Answer: Absolute positions an element relative to its nearest positioned ancestor, fixed keeps it in place relative to the viewport, and relative shifts it based on its normal position.
