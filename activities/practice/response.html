Questions:

  1. Why is the viewport meta tag important?

    2. Answer:It allows to html to be displayed correctly in mobile devices. The viewport meta tag is important because it helps control how a webpage is displayed on different devices, especially on mobile screens.

  2. How can you adjust the CSS for printing?

    3. Answer:using the @media print rule. This allows you to define styles that apply only when the page is printed.

  3. When would you want to adjust the CSS for printing?

    4. Answer: Adjusting CSS for printing is useful when creating printer-friendly versions of webpages by improving readability and removing interactive or irrelevant elements.
