Internet

Web

Definition: A web is a compose of webpages and sites that can be browse in Chrome, Firefox or Safari.

How are the Internet and Web different?

Answer: Internet connects one device to another or more. In simple words, the internet is the way of communicating information globally. On the otherhand, web is the one who use the internet to send and receive data.

ASCII and Unicode

Definition: ASCII stands for American Standard Code for Information Interchange its purpose is to represents basic characters into numbers from 0 to 127. While Unicode, is the advance version of ASCII in which it represents characters from many languages and symbols.

What is UTF-8?

Answer: It encodes the characters for the Unicode. This process of coding makes it effecient because it use fewer bytes for simple characters.

URL

Definition: URL stands for Uniform Resource Locator which means it is the address of the wed that you type to access a specific website. For example, www.youtube.com.

Domain Name

Definition: Domain Name are customized name of an IP address to easily access and to remember website. IP address are tend to be long and if the purpose of the website is to be on the internet and be known to people having a domain name would be the effecient and effective.

HTTP

Definition: Stands for HyperText Transfer Protocol. It is a set of regulation or rules that is used by browsers and servers to exchange data.

HTTPS

Definition:Stands for HyperText Transfer Protocol Secure. A safer and secure version of HTTP that ecrypts the data between the site and the browser.

IP Address

Definition:IP Address are unique number assigned to every device on the internet. It identifies a device on the internet.

DNS

Definition: DNS which stands for Domain Name System. This system turns a website name into an IP Address to locate them.

W3C Recommendations

Definition:These are the official rules created by the W3C which stands for World Wide Web Consortium to double check the sites and web technologies works well for everyone.

XML, XHTML, HTML, HTML5

Definition: XML is stands for eXtensible Markup Language. Its purpose is store and share date in an organized order. Then, XHTML is stands for eXtensible HyperText Markup Language. This is a way to write web pages in a stricter way with more rules. Then, HTML whhich stands for HyperText Markup Language. This is the language used to create web pages. Lastly, HTML5. This is the latest version of HTML with upgraded features for advance websites.
